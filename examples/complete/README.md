# Complete

This example demonstrates how to use the `terraform-alicloud-ecs-deploy-deepsite-application` module to create an ECS instance on Alibaba Cloud and deploy the DeepSite application.

## Usage

To run this example, execute the following commands:

```bash
$ terraform init
$ terraform plan
$ terraform apply
$ terraform destroy
```

**Note:** The resources created by this example cost money. Please be aware of the resources before applying and destroy them after you are done testing.
